# AI Chatbot Mentor

## Overview
AI-powered domain-specific learning assistant using Gemini, Hugging Face, LangChain, and Streamlit.

## Features
- Multi-domain learning support
- Topic-restricted AI responses
- Multi-turn conversation memory
- Chat export
- LLM routing
- Real-time AI responses

## Technologies Used
- Python
- Streamlit
- LangChain
- Gemini API
- Hugging Face

## Run Project

pip install -r requirements.txt

streamlit run app.py
